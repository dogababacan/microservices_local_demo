  • Port: 3005 (or similar)
  • Style: Global Consumer
  • Main job: Follows and logs every event in the project
  • Consumes: # (RabbitMQ wildcard for all events)
  • Queue (Inbox): analytics_service_events_queue
  • Notice: Perfect example of the "Wiretap" or "Fan-out" pattern. It collects data completely silently without breaking the main flow.
  • Ports: 5672 (AMQP), 15672 (UI)
  • Main job: Message Broker / Notice Board
  • Exchange: ecommerce_events
  • Notice: RabbitMQ decouples services completely.
